# HTML Only Portfolio

This is a solution to the "HTML Only Portfolio" project challenge on [roadmap.sh]([https://roadmap.sh/](https://roadmap.sh/projects/basic-html-website)). The primary goal of this project is to build a well-structured, multi-page website using purely semantic HTML, without relying on any CSS for styling.

## 🎯 Project Goals Achieved

- **Multi-page Architecture:** Created a functional website structure with multiple pages navigating seamlessly.
- **Semantic HTML:** Utilized proper HTML tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`) to ensure accessibility and meaningful document structure.
- **SEO Optimization:** Added relevant SEO meta tags (`description`, `keywords`, `author`) in the `<head>` section of every page.
- **HTML Forms:** Built a structured contact form utilizing appropriate input types (`text`, `email`, `select`), `<textarea>`, and `<label>` associations.
- **Future-Proofing:** Laid down a clean structural foundation that is ready to easily adopt CSS styling in the next phase of development.

## 📂 Project Structure

The repository consists of 4 interconnected HTML files:
- `index.html` - The main homepage serving as the primary resume/portfolio view.
- `projects.html` - A dedicated page listing completed and ongoing projects.
- `articles.html` - A page structured to host written articles or blog posts.
- `contact.html` - A page featuring a user inquiry form.
